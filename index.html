Explicações: do Trabalho:

Neste trabalho das quatro possíveis operações com matrizes, Optei pelas operações de adição e multiplicação. Para fazer isso, desenvolvi um código em JavaScript que cria duas matrizes de 3 por 3 com valores inteiros aleatórios que variam entre 0 e 100. Em seguida, os valores das matrizes são tranferidas para uma tabela e depois são executadas as operações de adição e multiplicação. Os resultados são mostrados na página HTML e também no console do navegador.

A função "somarMatrizes" faz a soma de matrizes, essa função usa duas matrizes como argumentos e retorna uma nova matriz na qual que é a soma das duas matrizes que recebeu. Para somar matrizes é necessário encontrar elementos correspondente de cada matriz exemplo: X11 e y11 ( 11 = linha 1 e coluna 1) e dps soma os valores.

A função "multiplicarMatrizes" faz a multiplicação de matrizes. Essa função usa duas matrizes como argumentos e depois retorna uma nova matriz que é o produto das duas matrizes que recebeu. Para multiplicar as matrizes vou dar um exemplo: você ja tem as matrizes X e Y e para encontar a Matriz C primeiro tera que calcular o C11(primeira linha, primeira coluna de C),você multiplica cada elemento da primeira linha de X pelo elemento correspondente da primeira coluna de Y e soma os resultados: C11 = X11*Y11 + X12*Y21 + X13*Y31 ;;; C21 = X21*Y11 + X22*Y21 + X23*Y31; e assim sequencialmente.
